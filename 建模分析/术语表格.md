# 术语表格

本文档用于保证数学建模分析、代码实现和论文撰写阶段术语一致。

## 一、核心概念术语

| 中文名称 | 英文名称 | 缩写/符号 | 定义/说明 |
|---|---|---|---|
| 无人机 | Unmanned Aerial Vehicle | UAV | 执行侦察任务的飞行平台 |
| 无人机群 | UAV Swarm | - | 多架无人机组成的协同系统 |
| 战场风险场 | Battlefield Risk Field | $p_{ij}$ | 网格位置对应的综合被发现或被击落风险 |
| 基础风险概率 | Base Risk Probability | $p^{base}_{ij}$ | 附件一给出的网格基础被发现概率 |
| 防空附加风险 | Air-defense Additional Risk | $p^{air}_{ij,k}$ | 防空火力点对周边网格造成的高斯附加风险 |
| 高价值目标 | High-value Target | HVT | 具有较高侦察价值的区域或目标簇 |
| 目标簇 | Target Cluster | - | 相邻高价值网格合并形成的任务目标 |
| 目标权重 | Target Weight | $w_q$ | 目标的重要程度或价值 |
| 起降点 | Depot/Base | $b_u$ | 无人机出发和返回的位置 |
| 补给点 | Replenishment Point | - | 可供无人机充电、维修或返航降落的位置 |
| 信息孤岛 | Information Island | - | 通信中断后与主通信网络断开的无人机或子群 |
| 置信状态 | Belief State | $B_u(t)$ | 无人机基于最近同步信息和自身观测形成的局部态势估计 |

## 二、数学符号术语

| 符号 | 含义 | 单位/类型 | 英文名称 |
|---|---|---|---|
| $\Omega$ | 战场网格集合 | 集合 | grid set |
| $(i,j)$ | 网格行列编号 | - | row-column index |
| $(x,y)$ | 平面坐标 | km 或网格单位 | coordinate |
| $U$ | 无人机集合 | 集合 | UAV set |
| $M$ | 侦察目标集合 | 集合 | target set |
| $n$ | 无人机数量 | 架 | number of UAVs |
| $m$ | 目标数量 | 个 | number of targets |
| $T_u^{\max}$ | 无人机 $u$ 最大续航时间 | min | maximum flight time |
| $T_u^{rem}$ | 无人机 $u$ 剩余航时 | min | remaining flight time |
| $T_u^{home}$ | 无人机 $u$ 安全返航所需时间 | min | return-to-base time |
| $v_u$ | 无人机 $u$ 最大飞行速度 | km/min | maximum speed |
| $r_u^s$ | 无人机 $u$ 传感器探测半径 | km | sensor range |
| $r_u^c$ | 无人机 $u$ 通信半径 | km | communication range |
| $\eta_u$ | 无人机 $u$ 能耗参数 | %/min 或题设单位 | fuel consumption |
| $status_u$ | 无人机 $u$ 状态 | 0/1 | UAV status |
| $P_u$ | 无人机 $u$ 的飞行路径 | 节点序列 | path |
| $S(P)$ | 路径生存概率 | $[0,1]$ | survival probability |
| $R(P)$ | 路径累计风险 | $[0,1]$ | cumulative risk |
| $C_R(P)$ | 路径对数风险代价 | 非负实数 | log-risk cost |
| $P_{safe}$ | 风险禁入阈值 | $[0,1]$ | safe risk threshold |
| $\tau_q$ | 目标 $q$ 识别停留时间 | min | service time |
| $d_{\min}$ | 多机安全距离 | km | safety distance |
| $H$ | 滚动预测窗口 | min 或步数 | prediction horizon |
| $Conf(\Delta t)$ | 信息置信度 | $[0,1]$ | confidence |

## 三、决策变量

| 变量 | 含义 | 类型 |
|---|---|---|
| $z_q$ | 目标 $q$ 是否被侦察完成 | 0-1 变量 |
| $y_{uq}$ | 目标 $q$ 是否分配给无人机 $u$ | 0-1 变量 |
| $x_{uv}$ | 单机路径中是否从节点 $u$ 到节点 $v$ | 0-1 变量 |
| $x_{uij}$ | 无人机 $u$ 是否从节点 $i$ 飞到节点 $j$ | 0-1 变量 |
| $t_{uq}$ | 无人机 $u$ 到达目标 $q$ 的时间 | 连续变量 |
| $\pi_{t:t+H}$ | 动态重规划窗口内的策略序列 | 策略变量 |

## 六、评价指标术语

| 中文指标 | 符号 | 定义/公式 |
|---|---|---|
| 总侦察收益 | $B$ | 被完成目标权重之和或期望收益 |
| 期望收益 | $\mathbb{E}[W]$ | $\sum_h w_{q_h}S_h$ |
| 任务完成率 | $\phi$ | 完成目标价值与总目标价值之比 |
| 总航时 | $T$ | 所有无人机航时之和或最大完成时间 |
| 总能耗 | $E$ | 所有无人机能耗之和 |
| 总风险代价 | $C_R$ | 所有路径对数风险代价之和 |
| 综合效能 | $J$ | 收益、时间、风险、能耗、均衡和鲁棒性的归一化加权值 |
| 负载均衡度 | $L$ | $1-\operatorname{std}(T_u)/(\operatorname{mean}(T_u)+\epsilon)$ |
| 通信连通率 | $\rho_c$ | 通信图保持连通的时间比例 |
| 协同鲁棒性 | $\rho$ | 战损后收益与基准收益之比 |
| 参数敏感度 | $S_\theta$ | $(\Delta J/J)/(\Delta\theta/\theta)$ |
| 投标收益 | $Bid_{uq}$ | 无人机 $u$ 对目标 $q$ 的边际收益评价 |
