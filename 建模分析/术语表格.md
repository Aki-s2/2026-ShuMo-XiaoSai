# 术语表格

## 一、核心概念术语

| 中文名称 | 英文名称 | 缩写/符号 | 定义 |
|---|---|---|---|
| 无人机 | Unmanned Aerial Vehicle | UAV | 执行侦察任务的飞行平台 |
| 无人机群 | UAV Swarm | - | 多架无人机组成的协同系统 |
| 战场风险场 | Battlefield Risk Field | $p_{ij}$ | 网格位置对应的综合风险概率 |
| 基础风险概率 | Base Risk Probability | $p^{base}_{ij}$ | 附件给出的网格基础被发现概率 |
| 防空附加风险 | Air-defense Additional Risk | $p^{air}_{ij,k}$ | 防空火力点对周边网格造成的附加风险 |
| 高价值目标 | High-value Target | HVT | 具有较高侦察价值的区域或目标簇 |
| 目标权重 | Target Weight | $w_q$ | 目标的重要程度或价值 |
| 目标簇 | Target Cluster | - | 相邻高价值网格合并形成的任务目标 |
| 起降点 | Depot/Base | $b_u$ | 无人机出发和返回的位置 |
| 补给点 | Replenishment Point | - | 可供无人机充电或维修的位置 |

## 二、数学符号

| 符号 | 含义 | 单位/类型 |
|---|---|---|
| $\Omega$ | 战场网格集合 | 集合 |
| $(i,j)$ | 网格行列编号 | - |
| $(x,y)$ | 平面坐标 | km 或网格单位 |
| $U$ | 无人机集合 | 集合 |
| $M$ | 侦察目标集合 | 集合 |
| $n$ | 无人机数量 | 架 |
| $m$ | 目标数量 | 个 |
| $T_u^{\max}$ | 无人机 $u$ 最大续航时间 | 时间 |
| $v_u$ | 无人机 $u$ 最大速度 | km/时间 |
| $r_u^s$ | 无人机 $u$ 传感器探测半径 | km |
| $r_u^c$ | 无人机 $u$ 通信半径 | km |
| $\eta_u$ | 无人机 $u$ 能耗参数 | 能耗/距离 |
| $status_u$ | 无人机 $u$ 状态 | 0 或 1 |
| $P_u$ | 无人机 $u$ 的飞行路径 | 节点序列 |
| $S(P)$ | 路径生存概率 | $[0,1]$ |
| $R(P)$ | 路径累计风险 | $[0,1]$ |
| $C_R(P)$ | 路径对数风险代价 | 非负实数 |

## 三、决策变量

| 变量 | 含义 | 类型 |
|---|---|---|
| $z_q$ | 目标 $q$ 是否被侦察 | 0-1 变量 |
| $y_{uq}$ | 目标 $q$ 是否分配给无人机 $u$ | 0-1 变量 |
| $x_{uv}$ | 单机路径中是否从节点 $u$ 到节点 $v$ | 0-1 变量 |
| $x_{uij}$ | 无人机 $u$ 是否从节点 $i$ 飞到节点 $j$ | 0-1 变量 |
| $t_{uq}$ | 无人机 $u$ 到达目标 $q$ 的时间 | 连续变量 |

## 五、评价指标

| 指标 | 符号 | 定义 |
|---|---|---|
| 总侦察收益 | $B$ | 被完成目标权重之和 |
| 任务完成率 | $\phi$ | 完成目标价值与总目标价值之比 |
| 总航时 | $T$ | 所有无人机航时之和 |
| 总能耗 | $E$ | 所有无人机能耗之和 |
| 总风险代价 | $C_R$ | 所有路径对数风险代价之和 |
| 协同效率 | $\mathcal{E}$ | 单位代价获得的侦察收益 |
| 鲁棒性 | $\rho$ | 无人机损毁后效率保持率 |
| 参数敏感度 | $S_\theta$ | 指标相对变化与参数相对变化之比 |
| 负载均衡度 | $L$ | 多架无人机任务时长或任务量的均衡程度 |
| 置信度 | $Conf(\Delta t)$ | 信息随延迟增加后的可信程度 |
| 投标收益 | $Bid_{uq}$ | 无人机 $u$ 对目标 $q$ 的边际收益评价 |
