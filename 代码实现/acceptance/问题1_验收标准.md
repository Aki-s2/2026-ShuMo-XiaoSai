# 问题1代码验收标准

本文档用于判断问题1“单无人机静态风险约束路径规划”的代码结果是否达到可写入论文的水平。参考优秀优化类数模论文的处理方式，问题1不强制做正态性检验、显著性检验等统计推断；验收重点应放在“模型可行、算法有效、对比充分、敏感性可解释、图表能支撑结论”五类证据上。

## 一、优秀论文启发下的验收框架

优秀优化类论文通常不会只给一条最优路径，而是同时给出以下证据：

| 论文常见做法 | 对问题1的转化要求 |
|---|---|
| 明确模型输入和约束边界 | 检查附件读取、有效网格范围、火力点数量、目标簇数量和空间覆盖范围 |
| 给出可行性验证 | 检查起终点闭合、续航约束、目标不重复、风险概率范围 |
| 与基准策略对比 | 至少与最近邻基准对比期望收益，证明贪心插入+2-opt不是随意路线 |
| 给出敏感性分析 | 检查网格尺度变化后候选数、完成数、收益和风险的变化趋势 |
| 用图表解释结果 | 用热力图、路径图、分布图、散点图说明目标分布、风险偏好和路径选择 |
| 说明异常结果 | 若生存概率很低，不能直接判定代码错误，应解释为高收益与高风险权衡，并在论文中说明风险偏好 |

## 二、数据与建模前提验收

| 验收项 | 合格标准 | 说明 |
|---|---|---|
| 附件读取 | 基础风险、防空火力、价值图、无人机参数均成功读取 | 代码应输出维度和关键数量 |
| 公共区域 | 使用 `400 x 234` 公共有效区域 | 与建模分析报告一致 |
| 防空火力点 | 检出 7 个火力点 | 若不是 7 个，说明附件读取或坐标解析异常 |
| 目标簇数量 | `value > 0.1` 时目标簇数量应为 384 左右 | 当前附件和阈值下的基准值为 384 |
| 目标簇空间范围 | `x` 覆盖约 `[5,230]`，`y` 覆盖约 `[3,399]` | 若只集中在左上角，说明目标簇提取或绘图坐标有问题 |
| 权重分布 | 最大权重显著高于中位数 | 高价值目标应呈长尾分布 |
| 风险分布 | 风险值在 `[0,1)` 内，且不应整体接近 1 | 若大量为 1，风险合成过强 |

## 三、模型可行性验收

| 验收项 | 合格标准 | 说明 |
|---|---|---|
| 起终点闭合 | 路径从 `(0,0)` 出发并返回 `(0,0)` | 问题1闭合巡航要求 |
| 单机平台 | 采用 UAV-04 作为代表单机平台 | 题目未限定编号，UAV-04 起点为 `(0,0)` 且续航更适合展示模型 |
| 续航约束 | `time <= max_time` | 使用最终 A* 可执行路径的航时验收 |
| 完成目标 | 完成目标数 `target_count > 0` | 若为 0，候选筛选或尺度口径异常 |
| 候选合法 | `candidate_count >= target_count` | 路径访问目标必须来自候选集合 |
| 目标不重复 | 路径序列中目标编号不重复 | 对应 OP 节点选择约束 |
| 风险概率 | `0 <= survival <= 1` | 生存概率必须在数学上合法 |
| 评价口径一致 | 结果指标按 A* 可执行路径计算 | 避免“图上绕行、数值按直线”的口径冲突 |

## 四、算法有效性验收

| 验收项 | 合格标准 | 说明 |
|---|---|---|
| 基准对比 | 启发式结果期望收益不低于最近邻基准 | 优秀论文常用“与简单规则对比”证明算法有效 |
| 局部优化有效 | 2-opt 后路径不劣于插入初始解 | 可接受相等，不允许变差 |
| 约束优先 | 所有输出路径均满足续航约束 | 不允许为了收益破坏约束 |
| 收益解释 | 给出权重收益和期望收益 | 区分“侦察价值”和“风险折减后价值” |
| 风险解释 | 给出风险代价和生存概率 | 生存概率低时标记为“需复核”，但不等于代码错误 |

## 五、敏感性与稳健性验收

| 验收项 | 合格标准 | 说明 |
|---|---|---|
| 网格尺度敏感性 | `cell_size_km` 增大时，候选目标数通常不增加 | 若反常，需要解释尺度口径或候选筛选 |
| 敏感性可行域 | 敏感性分析中的各测试点必须标明 `feasible` 状态 | 若出现 `feasible=False`，不能直接记为通过，应标记为“超出可行域”或“需复核” |
| 收益敏感性 | 不同尺度下期望收益变化可解释 | 用于论文讨论模型对空间尺度的依赖 |
| 风险敏感性 | 生存概率和风险代价随尺度变化有记录 | 用于解释风险惩罚的影响 |
| 可复现实验 | 每次运行输出同名结果表和图表 | 便于论文写作和版本对比 |

## 七、验收结论规则

| 结论 | 判定标准 |
|---|---|
| `通过` | 数据读取、核心约束、基准对比、评价口径、图表清单均合格，且风险指标无严重异常 |
| `需复核` | 核心约束满足，但存在生存概率过低、敏感性点落出可行域、敏感性趋势不稳定、收益-风险权衡过激等需要论文解释的问题 |
| `不通过` | 续航、起终点、目标重复、目标簇范围、风险概率合法性等任一核心硬约束失败 |

## 八、当前问题1重点关注项

当前问题1最容易误判的地方有两个：

1. 目标簇看起来集中在左上角时，要先看全局目标分布图和可达范围图，不应只看局部放大的路径图。
2. 路径评价必须以 A* 可执行避险路径为准；直线段只作为规划近似，不作为最终验收口径。
